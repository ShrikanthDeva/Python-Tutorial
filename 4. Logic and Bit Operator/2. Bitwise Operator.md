# Bitwise Operators

## Conjuction Operator  ``` & ```
  ```.py
   >>> 1 & 1
   >>> 1 & 0
   >>> 0 & 1
   >>> 0 & 0
  ```
  + O/P-

    > 1

    > 0

    > 0

    > 0
---

## Disjuction Operator  ``` | ```
  ```.py
   >>> 1 | 1
   >>> 1 | 0
   >>> 0 | 1
   >>> 0 | 0
  ```
  + O/P-
    > 1

    > 1

    > 1

    > 0

---

## Negation Operator  ``` ~ ```
  ```.py
   >>> ~ 1
   >>> ~ 0
  ```
  + O/P-
    > 0

    > 1

---

## Exclusive Operator  ``` ^ ```
  ```.py
   >>> 1 ^ 1
   >>> 1 ^ 0
   >>> 0 ^ 1
   >>> 0 ^ 0
  ```
  + O/P-
    > 0

    > 1

    > 1

    > 0

---


## BIT SHIFTING
  
  + Right Shift  ``` >> ```
    + Shifts the bits once to the right
    + Eg - 
      ```.py
       print( 22 >> 1 )
      ```
    + O/P-
      > 11

    + ```22 -> 0b00010110  >>  1```   ----->  ```11 -> 0b00001011```
    
+ LEFT Shift  ``` << ```
  + Shifts the bits once to the left
  + Eg - 
    ```.py
     print( 22 << 1 )
    ```
  + O/P-
    > 44

  + ```22 -> 0b00010110  <<  1```   ----->  ```44 -> 0b00101100```
