# String Operations

+ To Concatinate two strings use ```+``` operator
  + Eg -
    ```.py
      S1 = "Hello" + " " + "World!"
      print(S1)
    ```
  + O/P - 
    ```.py
      Hello World!
    ```
+ To repeat the string for certain amount of time use ```*``` operator.
  + Eg -
    ```.py
      S1 = "Hello World! " *2
      print(S1)
    ```
  + O/P - 
    ```.py
      Hello World!Hello World!
    ```
 + To type cast to string use ``` str() ``` function.
